Introduction

This documents the software for application note.

Compilation Info

This software was compiled with IAR Embedded Workbench, 5.30.6. To use GCC the source have to be modified.

To make project in IAR EWAVR:
Add the .c files, for the given example, to your project.Use device xmega256A3B, enable bit definitions in I/O include files, optimization low for debug target and high for release, output format: ubrof8 for Debug and intel_extended for Release, select Normal DLIB as library.

Device Info

All XMEGA devices with the targeted module can be used. The example is written for ATxmega256A3B.
