Introduction

This service provides support for accessing, configuring, enabling and disabling internal clocks on AVR devices.

Clock Examples

  • Constant-frequency LED blinking using a fixed clock configuration.
  • Clock output on a pin.
  • Cycling through various clock sources at run time.
